# Context - SaveTheBus

## Current State

The SaveTheBus project is a fully functional single-page React application for generating objection letters against Rule 288-A in Tamil Nadu. The application is production-ready and deployed via Vercel.

## Recent Changes

No recent changes have been documented. The project appears to be in a stable state with all core features implemented.

## Current Focus

The application is currently focused on:
- Helping citizens submit objections before the January 7, 2026 deadline
- Providing AI-powered letter generation with fallback templates
- Supporting both English and Tamil languages

## Next Steps

Potential areas for future development:
- Analytics integration to track objection submissions
- Additional language support if needed
- Campaign metrics dashboard
- Email delivery tracking (currently uses mailto links)
- Mobile app version for broader reach

## Deployment

- Platform: Vercel
- Environment variable required: `GEMINI_API_KEY` for Google Gemini AI
- Development server: `npm run dev` (runs on port 3000)
- Build command: `npm run build`
